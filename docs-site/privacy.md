# Privacy Policy

b-board is a client-side JavaScript library. It does not collect, transmit, or store any personal data.

## What we collect

**Nothing.** The library runs entirely in the user's browser. No telemetry, no analytics, no network requests are made by b-board itself.

## This documentation site

This docs site is a static site hosted on GitHub Pages. It does not use cookies, tracking pixels, or analytics scripts.

## Contact

If you have a privacy concern, open an issue at [github.com/SoftZenIT/b-board/issues](https://github.com/SoftZenIT/b-board/issues).
