# Politique de confidentialité

b-board est une bibliothèque JavaScript côté client. Elle ne collecte, ne transmet ni ne stocke aucune donnée personnelle.

## Ce que nous collectons

**Rien.** La bibliothèque s'exécute entièrement dans le navigateur de l'utilisateur. b-board n'effectue aucune télémétrie, aucun suivi, aucune requête réseau.

## Ce site de documentation

Ce site est un site statique hébergé sur GitHub Pages. Il n'utilise ni cookies, ni pixels de suivi, ni scripts d'analyse.

## Contact

Pour toute préoccupation relative à la confidentialité, ouvrez un ticket sur [github.com/SoftZenIT/b-board/issues](https://github.com/SoftZenIT/b-board/issues).
